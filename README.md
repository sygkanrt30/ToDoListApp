# ToDoList Application

ToDoList is a standard CRUD (Create, Read, Update, Delete) application designed for managing your tasks efficiently. It can serve as a supportive tool for your time management system.

## Technologies Used

- **JavaCore**
- **PostgreSQL**
- **JavaFX**
- **JDBC**

## Features

The ToDoList application offers a variety of features to help you manage your tasks effectively:

- **User Registration**: Register in the system with email verification via a confirmation code sent to your email.
- **Authentication**: Securely log in to your account.
- **Task Management**:
  - Create tasks with the ability to set priorities and deadlines.
  - Rename existing tasks.
  - Mark tasks as completed or not completed.
  - Complete all tasks in one go.
  - Delete individual tasks or all tasks at once.
  - Remove all completed tasks.
- **Navigation**: Easily navigate back to previous screens at any stage of the process.
- **Logout**: Option to log out of your account.

## Getting Started

To get started with the ToDoList application, ensure you have the necessary technologies installed:

1. **Java Development Kit (JDK 23)**
2. **PostgreSQL Database**
3. **JavaFX Libraries**

Clone the repository and follow the setup instructions to run the application locally.
